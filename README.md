# Dove & Smile Studio 

A responsive dental clinic website built using HTML, CSS, and JavaScript.  
The project includes multiple pages, reusable components, and a simple authentication system using Local Storage.

---

##  Project Overview

Dove & Smile Studio is a front-end website for a dental clinic.  
It provides information about the clinic, dental services, team members, pricing plans, and contact details.  
The project also includes login and registration pages for educational purposes.

---

##  Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- Local Storage  
- Swiper.js

##  Pages Description

###  Home Page (`index.html`)
- Main landing page
- Introduces the clinic and its services
- Loads navbar and footer dynamically

### ℹ About Page (`about.html`)
- Information about the clinic
- Mission, vision, and values

###  Team Page (`team.html`)
- Displays doctors and staff members
- Includes interactive slider

###  Plans Page (`plans.html`)
- Shows pricing plans and services

###  Contact Page (`contact.html`)
- Clinic contact details and location

###  Login Page (`login.html`)
- User authentication using Local Storage

###  Register Page (`register.html`)
- Create a new user account
- Stores user data in Local Storage

---

##  JavaScript Files

### `include.js`
- Loads navbar and footer dynamically
- Prevents repeating HTML code

### `login.js`
- Handles login validation
- Compares user data with Local Storage

### `register.js`
- Handles user registration
- Saves user data to Local Storage

### `script.js`
- Controls UI interactions and animations

3. No backend or database required
